# Little English Quest

A touch-first English learning app for children ages 3 to 6.

## What It Includes

- Theme quests for animals, fruits, and food
- Learn, listen, speak, and reward steps for each word
- Explore mode with tappable vocabulary cards
- Local progress and sticker rewards
- PWA support for installable use on tablets and desktops

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Test

```bash
npm test
npm run e2e
```
